# 更新日志

所有重要更改都会记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2026-06-24

### 新增

- 恶搞虚假下载模式（默认模式）
  - 进入世界自动弹出仿高危系统警告弹窗
  - 红色警告边框 + 恐怖中文文案轮播
  - 进度条缓慢递增，5%概率回退模拟网络卡顿
  - 进度永远锁定在99%，不会到100%
  - 支持关闭按钮和ESC键关闭

- 真实下载模式（配置开启）
  - 异步后台下载指定URL文件
  - 实时显示下载进度、速度、剩余时间
  - 下载完成后自动打开安装包
  - 完善的错误提示（网络超时、权限不足等）

- 配置系统
  - 自动生成默认配置文件
  - 配置损坏时自动恢复默认值
  - 支持自定义下载链接

- 本地化支持
  - 中文（简体）
  - 英文

### 技术实现

- 基于 Fabric API 的 `ClientPlayConnectionEvents.JOIN` 事件监听
- 使用 `CompletableFuture` 异步下载，不阻塞游戏主线程
- `AtomicLong` / `AtomicBoolean` 保证线程安全
- Kotlin 空安全语法 + 全面异常捕获
